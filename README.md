## Focus
The focus of this project is to build out an analysis of schools in the United States

### Process
We began by compiling the data from nces.ed.gov.
Data used in the project were collected from the following links, renamed, and placed in the data folder
- [US Schools Characteristics](https://nces.ed.gov/ccd/Data/zip/ccd_sch_129_2425_w_1a_073025.zip)
- [US Schools Details 24-25](https://nces.ed.gov/ccd/Data/zip/ccd_sch_029_2425_w_1a_073025.zip)
- [US Schools Demographics 24-25](https://nces.ed.gov/ccd/Data/zip/ccd_lea_052_2425_l_1a_073025.zip)
- [Private Schools Data 21-22](https://nces.ed.gov/surveys/pss/zip/pss2122_pu_csv.zip)
